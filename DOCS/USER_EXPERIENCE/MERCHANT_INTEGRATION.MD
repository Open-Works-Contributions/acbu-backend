# Merchant Integration

## Overview

ACBU enables merchants to accept payments from customers across 10 African countries without forex exposure.

## Value Proposition for Merchants

### Benefits
- **Accept one currency (ACBU)** from all 10 countries
- **Lower fees** (0.2% vs 2-3% card fees)
- **No currency conversion** needed
- **Simplified accounting**
- **Instant settlement** (3-5 seconds)
- **No forex risk** exposure

## Target Merchants

### E-commerce Platforms
- Jumia
- Konga
- Takealot
- Regional marketplaces

### Service Providers
- Hotels
- Airlines
- Restaurants
- Retail stores

### Payment Processors
- POS systems (Yoco, Amplify)
- Online payment gateways

## Integration Methods

### 1. Payment Gateway SDK
- JavaScript/React SDK
- Easy integration for web merchants
- QR code generation
- Payment status callbacks

### 2. QR Code Payments
- Works offline
- Mobile wallet scanning
- Instant settlement
- Low infrastructure requirements

### 3. POS Integration
- Compatible with Yoco, Amplify
- In-store payments
- Receipt generation
- Transaction history

### 4. Mobile Money Interoperability
- Integration with mobile money systems
- Accept payments via mobile money
- Convert to ACBU automatically
- Seamless user experience

## Merchant Onboarding

### Requirements
1. Business registration
2. KYC verification
3. Bank account or mobile money account
4. Integration setup (SDK or API)

### Process
1. Merchant applies for ACBU acceptance
2. KYC verification
3. Integration setup (technical support provided)
4. Test transactions
5. Go live

## Settlement Options

### Option 1: ACBU Balance
- Merchant receives ACBU
- Can hold ACBU for future use
- No conversion needed
- Earn yield on ACBU balance (future feature)

### Option 2: Local Currency Withdrawal
- Merchant withdraws to local currency
- 0.2% merchant fee
- Instant settlement
- Funds appear in merchant account

### Option 3: Automatic Conversion
- Automatic conversion to local currency
- Daily batch settlement
- Reduced fees for high-volume merchants

## Merchant Dashboard

### Features
- Transaction history
- Settlement status
- Revenue analytics
- Multi-currency reporting
- Payment links generation
- QR code management

## Pricing

### Merchant Fees
- **Accept ACBU:** 0.2% per transaction
- **Withdraw to local currency:** 0.3% (if not included in merchant fee)
- **API access:** $500/month (premium tier)

### Comparison
- **Card payments:** 2-3% fees
- **ACBU payments:** 0.2% fees
- **Savings:** 90%+ reduction in fees

## Goals

### Phase 1: Payment Infrastructure (Year 1-2)
- **Goal:** 100,000 merchants accepting ACBU
- **Focus:** E-commerce platforms, major retailers
- **Incentive:** Lower fees than card payments (0.2% vs 2.5%)

### Phase 2: Financial Services (Year 2-3)
- ACBU savings products for merchants
- Merchant lending (ACBU-denominated)
- Enhanced analytics and reporting

### Phase 3: Ecosystem Lock-In (Year 3+)
- **Network Effect:**
  - "I accept customers from 10 countries"
  - "No forex risk, instant settlement"
  - "Pan-African customer base"
